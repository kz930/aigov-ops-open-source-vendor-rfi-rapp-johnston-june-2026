AiGovOps Foundation · Beacon × Umbrella

Open-source vendor RFI for AI governance that runs like engineering.

This repository packages a static webapp, deployment workflow, auditor-facing value model, and vendor sponsorship design so the Foundation can publish a neutral, evidence-oriented market surface instead of a static vendor list.

Open webapp Vendor guide Auditor value Vendor sponsorship Repo guide

What this gives auditors

Not just screenshots and claims. The design centers on portable decision artifacts, profile-based rationale, deterministic exports, and a future path to Beacon-signed evidence that procurement, audit, and engineering can all inspect.

What this gives vendors

A structured way to explain fit, gaps, interoperability, and sponsor participation without collapsing Foundation neutrality. Vendors can support the ecosystem while the Foundation preserves open evaluation logic.
