# Vendor Sponsorship Model

Safe-RFI uses neutral sponsorship governance.

## Purpose of sponsorship

Sponsorship helps fund open ecosystem surfaces, laboratory scenarios, connectors, and evidence tooling improvements.

## What sponsorship does not do

Sponsors do **not** buy:

- ranking position
- approval outcomes
- hidden weighting changes
- gap suppression

## Sponsorship tiers

1. **Community supporter**
2. **Lab sponsor**
3. **Integration sponsor**
4. **Evidence sponsor**

## Disclosure language

Safe-RFI displays sponsorship disclosures in public documentation and in the advisor UI.
All stack recommendations remain evidence-oriented and deterministically generated from visible criteria.
