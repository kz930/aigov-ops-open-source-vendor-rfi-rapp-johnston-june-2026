# Safe-RFI

Safe-RFI is the AIGovOps Foundation’s open-source, evidence-oriented vendor RFI experience for AI governance.

It is designed for transparent procurement and controls alignment workflows across security, audit, engineering, and policy teams.

## Program alignment

- **Beacon**: Safe-RFI emits receipt-ready decision artifacts and an unsigned receipt preview path.
- **Umbrella**: Safe-RFI supports policy and controls framing for deployment governance.
- **Lantern (future)**: Safe-RFI output shape is prepared for future diff-oriented alignment.

## GitHub Pages publishing

This repository includes a GitHub Actions Pages workflow that publishes static content from `webapp/` with no build system requirement.

## Sponsorship governance

Safe-RFI uses neutral sponsorship governance. Sponsors can fund ecosystem surfaces, labs, connectors, and evidence tooling, but cannot buy ranking, approval outcomes, hidden weighting changes, or gap suppression.
