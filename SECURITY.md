# Security Policy

## Supported versions

Safe-RFI is a static-site project. Security fixes are applied on the default branch and included in the next GitHub Pages deployment.

| Version | Supported |
| --- | --- |
| main | ✅ |

## Reporting a vulnerability

Please report suspected vulnerabilities through GitHub Security Advisories for this repository.

When submitting a report, include:

- clear reproduction steps
- affected file paths and line references
- expected versus observed behavior
- impact assessment for confidentiality, integrity, and availability

Please do not disclose exploit details publicly before maintainers have had a reasonable chance to investigate and patch.

## Scope notes

This repository ships static HTML, CSS, JavaScript, and project documentation only. No hosted API or server-side processing is included.
